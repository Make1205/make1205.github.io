1.2. Matlab数值数据

1.2.1. 数值数据类型

1.整型

无符号8位整数数据范围：00000000~11111111(0~$2^8-1$)
有符号8位整数数据范围：10000000~01111111($-2^7$~$2^7-1$)

2.浮点型

single函数：将其他类型数据转化为单精度。
double函数：将其他类型数据转化为双精度。
class函数：返回传入参数的数据类型。

3.复数型
复数型数据包括实部和虚部两个部分，实部和虚部默认为双精度，虚数单位用i或者j来表示。
real函数：返回实部
imag函数：返回虚部

1.2.3.2.3. 取整函数

取整的函数有fix、floor、ceil、round

fix是固定取靠近0的那个整数，也就是直接舍去小数
floor是向下取整，取小于等于整数的第一个整数
ceil是向上取整，取大于等于整数的第一个整数
round直接四舍五入。

1.5.1.2. 通过序号访问

在matlab中，矩阵元素按列存储，首先第一列，然后第二列……一直到最后一列
矩阵元素的序号就是矩阵元素在内存中的排列顺序

>> A=[1,2,3;4,5,6;7,8,9]

A =

     1     2     3
     4     5     6
     7     8     9

>> A(3)

ans =

     7

序号与下标一一对应，m×n的矩阵A:
A(i,j)=A((j-1)×m+i)

sub2ind函数：将矩阵中指定元素的行列下标转换成储存的序号

调用格式：D=sub2ind(S,I,J) D为序号数组 S为行数列数组成的向量，一般就是矩阵的size I转换矩阵行下标 J转换矩阵列下标

ind2sub函数：将矩阵元素的序号转换成对应下标

调用格式：[I J]=ind2sub(S,D) I行下标 J列下标 S行数和列数组成的向量 D序号

1.7.2.2. 字符串比较

有两种方式： 1.关系运算法比较，两个字符串里的每个字符一次按ASCII大小比较，结果是一个向量，元素0 or 1

>> 'abCD'>='ABcd'

ans =

  1×4 logical 数组

   1   1   0   0

2.字符串比较函数：
strcmp(s1,s2):相等1，否则0 strncmp(s1,s2,n):前n个是否相等，相等1，否则0 strcmpi(s1,s2):忽略大小写的strcmp strncmpi(s1,s2,n):忽略大小写的strncmp

1.7.2.3. 字符串查找替换

findstr(s1,s2):返回短字符串在长字符串中开始出现的位置 strrep(s1,s2,s3):将s1中所有子串s2替换成s3

> s='qwer1234'

s =

    'qwer1234'

>> ss='er'

ss =

    'er'

>> sss='ER'

sss =

    'ER'

>> findstr(s,ss)

ans =

     3

>> strrep(s,ss,sss)

ans =

    'qwER1234'
