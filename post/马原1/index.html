    1. 马克思主义是关于无产阶级和人类解放的科学

      1. 马克思主义基本原理的内涵
        1. 什么是马克思主义？
          1. 马克思主义只有马克思、恩格斯创立，并为后继者所不断发展的科学理论体系。
            1. 是关于自然、社会和人类思维发展一般规律的学说 （内容层面）
            2. 是关于社会主义必将代替资本主义，最终实现共产主义的学说 （奋斗目标）
            3. 是关于无产阶级解放，全人类解放和每个人自由而全面发展的学说
            4. 是指引人类创造美好生活的行动指南
          2. 三个基本组成部分：由一系列的基本理论、基本观点和基本方法构成的科学体系。
            1. 马克思主义哲学 （世界观和方法论）
            2. 马克思主义政治经济学 （应用）
            3. 科学社会主义 （目标） 核心部分
        2. 什么是马克思主义基本原理
          1. 市队马克思主义立场，观点和方法的集中概括，是马克思主义在其形成、发展和运用的过程中，经过实践反复检验而确立起来的具有普遍真理行的理论
          2. 体现了马克思主义的根本性质和整体特征，体现了马克思主义科学性和革命性的统一

    2. 世界的物质性及发展规律

      1. 世界多样性和物质统一性

        1. 一个问题：哲学基本问题
          1. 哲学基本问题的内涵（研究世界的本原）
            1. 恩格斯第一次明确提出“全部哲学的基本问题是思维和存在的关系问题”。对哲学基本问题的回答是解决其他一切哲学问题的前提和基础
            2. 包括两个方面的内容
              1. 其一，存在和思维究竟谁才是世界的本原，即物质和精神何者是第一性，何者第二性的问题。（物质=存在，意识=思维）根据对这一方面问题的不同回答，哲学可以划分为唯物主义和唯心主义两个对立的基本派别。唯物主义把世界本原归结为物质，主张物质第一性，意识第二性，意识是物质的产物，唯心主义反之。
              2. 其二，思维能否认识或正确认识存在，即思维和存在有无同一性的问题，根据对这一方面问题的不同回答，哲学又可以划分为可知论和不可知论。可知论认为世界是可以被认识的，思维和存在具有同一性，（不完全）不可知论认为世界不能被人缩认识或者不能被完全认识的，否认思维和存在的同一性（现象，本质）
          2. 辩证法和形而上学（研究世界如何存在）
            1. 根据对世界是怎样存在的问题，即世界上的事物是联系的还是独立的，是发展的还是静止的问题的不同回答形成了辩证法和形而上学两种不同观点。辩证法坚持用联系的、发展的观点看世界，认为发展的根本原因在于承认失误的内部矛盾（联系，全面，运动，发展）；形而上学坚持用鼓励的、静止的观点看问题，否认失误内部矛盾的存在和作用（鼓励，片面，静止）
          3. 只有既坚持唯物主义，又坚持辩证法，才能全面认识世界的本质和发展规律
          4. 只有达到唯物主义和辩证法的内在统一，才能有科学的、彻底的唯物主义和科学的彻底的辩证法
        2. 四个观
          1. 物质观（物质范畴及其理论意义）

            1. 物质概念：物质范畴值唯物主义哲学关于世界本原和统一性的最高抽象，是唯物主义世界观的基石，对于"物质"认识的深度和广度区分出三种唯物主义的形态:

              1. 古代朴素唯物主义:从具体物质形态出发说明世界,把物质等同于物质的具体形态,对世界物质性的理解和把握具有直观性和朴素行
              2. 近代形而上学唯物主义(机械):根据当时自然科学关于原子是物质最小单位的认识吧物质归结为原子,认为原子的属性就是一切物质形态的共同属性,虽然有一定科学依据,但是混淆了自然科学物质结构概念同哲学物质的范畴的区别,经不起自然科学发展大局检验,也经不起唯心主义的进攻,尤其不能说明社会生活的物质性.(半截子唯物主义=近代形而上学唯物主义)
              3. 现代辩证唯物主义:恩格斯从无限多样的物质现象中抽象出共同的本质,从哲学上作最高概括,认为物质无非是各种物的总和,而物质这个概念就是从这一总和中抽象出来的,指出了客观实在性是物质的本质,哲学物质概念与自然科学关于具体物质形态和物质结构的概念之间是共性哥惯性的关系.

            2. 20世纪初,列宁从物质与意识的关系上对物质概念做了全面的科学的规定:“物质是标志客观实在的哲学范畴,这种客观实在是人通过感觉感知的,他不依赖于我们的感觉而存在,为我们的感觉所复写、摄影、反映”

            3. 物质范畴是对物质世界的多样性和统一性所做的最高的哲学概括。物质的唯一特性是客观实在性，它存在于人的意识之外，并且为人的意识所反映。

            4. 确立物质概念的理论意义

              1. 坚持了唯物主义一元论，同唯心主义一元论和二元论划清了界限（一元论就是一个本原，二元论就是两个本原）
              2. 坚持能动的反映论和可知论，批判了不可知论
              3. 体现了唯物论和辩证法的统一，克服了形而上学唯物主义的缺陷
              4. 体现了唯物主义自然观与唯物主义历史观的统一，为彻底的唯物主义奠定了理论基础。
          2. 运动观（物质的根本属性）
            1. 物质和运动（2个关键词+2个错误）
              1. 运动是物质的存在方式和根本属性（固有属性），它涵盖了宇宙中发生的一切变化和过程
              2. 物质和运动是不可分割的：无运动的物质是形而上学，无物质的运动是唯心主义
            2. 运动和静止（2个关键词+1个关系）
              1. 静止是物质运动在于一定条件下的稳定状态，包括空间位置和根本性质暂时未变这样两种运动的特殊状态，静止也是运动的横梁尺度。
              2. 运动与静止的对立统一关系
                1. 运动是变动，无条件绝对的；静止是稳定有条件相对的（对立）
                2. 运动和静止相互依赖，相互渗透，相互包含，“动中有静，静中有动” （统一）
                3. 无条件的绝对运动和有条件的相对静止构成了事物的矛盾运动
          3. 时空观（物质的基本存在形式）
            1. 时间和空间是物质运动的存在形式：时间指物质运动的连续性，顺序性特定是一维；空间是物质运动的广延性，伸张性，特点是三维
            2. 物质运动与时间和空间的不可分割说明了时间和空间的客观性
            3. 时空的有限和无限：整个物质世界的时空是无限的，具体物质形态的时空是有限的；物质运动时间和空间的客观实在性是绝对的，物质运动时间和空间的而具体特性是相对的。
          4. 意识观
            1. 物质决定意识
            2. 意识对物质具有反作用意识依赖于物质并反作用于物质，这种反作用也是意识的能动作用，意识的能动作用是人的意识所特有的积极反映世界和改造世界的能力和活动，具体体现在：
              1. 意识活动具有目的性和计划性
              2. 意识活动具有创造性
              3. 意识具有指导实践，改造客观世界的作用
              4. 意识具有指导控制人的行为和生理活动的作用
            3. 意识与人工智能
              1. 人类意识是知情意的统一体
              2. 社会性是人的意识所固有的本质属性
              3. 人类的自然语言是思维的物质外壳和意识的显示形式
        3. 两个基本原理
          1. 主观能动性和客观规律性的辩证关系原理：如人与自然的关系，习近平指出，人类必须尊懂自然顺应自然保护自然，人类只有遵循自然规律才能有效防止在开发利用自然上走弯路。注意：
            1. 从实际出发努力认识和把握事物的发展规律
            2. 实践是发挥主观能动性的途径
            3. 发挥主观能动性还依赖于一定的物质条件和手段
          2. 世界的物质统一性原理及其意义
            1. 世界的物质统一性
              1. 世界是统一的，世界的本原是一个
              2. 世界的统一性在于他的物质性，世界统一的基础是物质，不是某种始基的物体
              3. 物质世界的统一性是多样性的统一而不是单一的无差别的统一
            2. 人类社会统一于物质
            3. 马克思主义关于世界物质统一性原理具有重大理论和实践意义

      2. 事物的普遍联系和变化发展

        1. 2个总特征：
          1. 第一个总特征：联系
            1. 内涵：联系是指事物内部各个要素之间和事物之间相互影响相互制约相互作用的关系（两个之间三个相互）
            2. 特点：
              1. 客观性：联系是事物本身固有的，坚持客观性要求我们从事物固有联系去把握事物，切忌主观随意，坚持客观性就是在联系的观点上坚持唯物论
              2. 普遍性：内部联系，外部联系以及普遍性联系
              3. 多样性：联系的内容和形式多样
              4. 条件性：条件是对事物存在和发展作用的诸要素总和
            3. 意义：马克思主义关于事物普遍联系额度原理要求人们善于分析事物的相互具体联系，确立整体性开放性观念，动态考察事物的普遍联系。要求我们正确认识和处理人与自然，人与人人与社会的相互关系。正确认识和处理中国特色社会主义事业中的重大关系，坚持统筹兼顾，促进经济社会协调持续发展，促进全面发展。世界角度顺应潮流。抓住机遇发展自己。
          2. 第二个总特征：发展
            1. 发展及其实质：
              1. 概括一切形式的变化就是运动，运动变化的基本趋势就是发展。
              2. 定义：发展是前进上升的运动，发展的实质是新事物产生旧事物灭亡
              3. 新生事物是不可战胜的
            2. 发展的过程性
              1. 发展是一个过程
              2. 意义：坚持事物发展是过程的看法，就是要用历史的眼光看问题，将一切事物看做变化发展的过程，观察他们的过去现在，预见未来
        2. 5个环节（联系和发展的基本环节）：世界普遍联系和永恒发展是通过一系列基本环节实现的
          1. 原因与结果
          2. 现象与本质
          3. 内容与形式
          4. 必然和偶然
          5. 可能与现实
        3. 3大规律（唯物辩证法的基本规则）唯物辩证法是一个完整的科学体系，包括一系列的基本规律和范畴
          1. 对立统一规律（矛盾规律）

          3. 否定之否定规律
        4. 辩证法的3个运用

    3. 实践与认识及其发展规律

      1. 认识与实践
        1. 实践的本质就是人类能动地改造世界的客观物质性活动
        2. 3个基本（实践的本质和基本结构）
          1. 实践的基本结构：人们实践活动是以改造客观世界为目的，主体客体之间你通过一定中介发生相互作用的客观过程，三项基本要素包括主体客体和中介。
          2. 实践的基本特征：具有客观性，能动性，历史性，分别是将实践与认识比较，人和动物比，过去和现在比
          3. 实践的基本类型
            1. 物质生产实践（基本）
            2. 社会政治实践
            3. 科学文化实践
        3. 1个决定作用（实践在认识中的决定作用）：实践是认识的基础，他对认识的决定作用表现在：
          1. 实践是认识来源
          2. 实践是认识发展的动力
          3. 实践是检验认识是否具有真理性的唯一标准
          4. 实践是认识的目的
        4. 3个论：认识是主体在实践基础上对客体的能动反映
          1. 唯心主义先验论
          2. 形而上学唯物主义直观反映论
          3. 辩证唯物主义能动反映论
        5. 1个过程
          1. 第一次飞跃：从实践到认识：感性认识到理性认识
          2. 第二次飞跃：从认识到实践：理性认识到实践
      2. 真理与价值
        1. 3个特征
          1. 真理的客观性
          2. 真理的绝对性和相对性及其辩证关系
            1. 真理的绝对性是指真理的内容表明了主客观统一的确定性和发展的无限性
